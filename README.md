# Three-Level Password System

## Overview

A Django-based authentication system that provides three levels of security:

1. Text Password Authentication
2. RGB Password Authentication
3. Image Password Authentication

## Features

* User Registration
* User Login
* RGB-Based Second Level Authentication
* Image-Based Third Level Authentication
* Dashboard Access After Successful Verification
* SQLite Database Storage
* Django Session Management

## Technologies Used

* Python
* Django
* HTML
* SQLite
* CSS

## Project Flow

Home Page → Register → Login → RGB Authentication → Image Authentication → Dashboard
